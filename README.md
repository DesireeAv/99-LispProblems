# 99-SchemeProblems
#### This repository has the pourpuse of studying the [99 Scheme Problems](https://www.ic.unicamp.br/~meidanis/courses/mc336/problemas-lisp/L-99_Ninety-Nine_Lisp_Problems.html)

## First problem:
```scheme
(define my-last
  (lambda(x)
    (cond ((null? (cdr x))
          (car x))
    (else
     (my-last(cdr x))))))
```
## Second problem:
```scheme
(define my-but-last
  (lambda(x)
    (cond ((null? (cdr (cdr x)))
          x)
      (else(my-but-last (cdr x))))))
```
